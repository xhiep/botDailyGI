bot.py — Telegram Bot điều khiển HoYoLab trực tiếp từ điện thoại
Chạy: python bot.py

Lệnh hỗ trợ:

/status — kiểm tra bot còn sống + trạng thái cookie

/uid — lấy UID và nickname tài khoản
  
/checkin — điểm danh HoYoLAB ngay

/redeem ABC — đổi 1 gift code

/redeemall — đổi toàn bộ code trong codes.txt

/characters — xem danh sách nhân vật

/resin — xem trạng thái nhựa hiện tại

/screenshot — chụp màn hình máy tính gửi lên

/ls [path] — xem danh sách file trong thư mục

/sendfile path — gửi file lên Telegram

/run cmd — chạy lệnh/chương trình trên máy

/fetchcodes — tìm code mới từ internet và redeem luôn

/shutdown — tắt máy tính

/restart — khởi động lại máy tính

/login — mở trình duyệt đăng nhập lại

/help — danh sách lệnh

CODE do AI tạo !!!
